# MindBridge-RoBERTa

Multilingual mental health detection for English, Hindi, and Hinglish social media text.

## Repository Contents

- `annotation/guidelines.pdf` — 24-page annotation guidelines
- `annotation/prompts.md` — GPT-4 and LLaMA-2 prompts used for silver labelling
- `data/corpus_statistics.json` — Dataset and DAPT corpus statistics

## Dataset
MindBridge-50K is available on HuggingFace: https://huggingface.co/datasets/mindbridge-50k

## Paper
MindBridge-RoBERTa: Multilingual Mental Health Detection for Indian Social Media
